# Roadmap

## Vision

Coach Automation Kit aims to become the **default open-source marketing automation platform** for coaches, consultants, and solopreneurs worldwide — replacing expensive SaaS tools with a free, self-hosted alternative.

---

## v1.3.0 — Q1 2026 (In Progress)

- [x] WhatsApp Business API integration
- [x] AI-powered email subject line generator
- [x] Multi-language email templates (Hindi, Spanish, Portuguese)
- [ ] Funnel builder with drag-and-drop editor
- [ ] Certificate of completion generator for course creators
- [ ] Booking system with calendar integration

## v1.4.0 — Q2 2026

- [ ] Multi-tenant SaaS mode (white-label for agencies)
- [ ] Stripe Connect for marketplace payments
- [ ] AI chatbot for lead qualification (Claude/Gemini)
- [ ] Advanced A/B testing for email sequences
- [ ] Webhook builder for custom integrations
- [ ] Mobile app (React Native)

## v2.0.0 — Q3 2026

- [ ] Plugin marketplace for community extensions
- [ ] Visual workflow builder (n8n-style)
- [ ] AI-powered analytics insights
- [ ] Self-hosted LLM integration (Ollama)
- [ ] Enterprise SSO (SAML/OIDC)
- [ ] Compliance toolkit (GDPR, CAN-SPAM, CASL)

## Long-term Vision

- Open-source alternative to HubSpot, ActiveCampaign, and GoHighLevel
- Community-driven plugin ecosystem
- 1,000+ coaches using it worldwide
- Localization for 10+ languages
- Integration with every major payment processor globally
